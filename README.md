Simple Login Page
This repository contains a simple login page created using HTML, CSS, and JavaScript. It provides a basic structure for a login interface with fields for username and password entry. Upon submission, the entered credentials are logged to the console for demonstration purposes.

Features
Clean and minimalist user interface.
Responsive design for seamless viewing across devices.
Easy to understand HTML, CSS, and JavaScript code structure.
Customizable styles and functionality to suit specific project requirements.
Usage
To use this login page, simply clone the repository to your local machine and open the index.html file in a web browser. You can customize the design, add validation logic, or integrate it into your web application as needed.
